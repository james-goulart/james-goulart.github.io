## Why the site has a copilot

Most portfolios are optimized for passive reading.

This one is also optimized for active exploration: a recruiter, hiring manager, or peer can ask direct questions about AI experience, leadership scope, case studies, or measurable outcomes.

The copilot is not the main proof. The underlying cases are.
