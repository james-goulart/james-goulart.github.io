# Homepage Phase 1 Copy Proposal

This document proposes the Phase 1 rewrite for the homepage above the fold.

**Shipped on the live site as** [Overview](https://james-goulart.github.io/overview.html) (site home; root `/` redirects there). Copilot lives at [copilot.html](https://james-goulart.github.io/copilot.html).

The goal is to make the site read less like a generic portfolio and more like the public profile of a Staff-level AI Product leader.

---

## 1. Hero headline

**Staff-level AI Product Leader**

Search, marketplaces, retrieval, trust, and LLM-native product experiences.

### Supporting line

I build products where user intent, system quality, and business outcomes have to work together.

### Optional alternative

I lead AI, search, and marketplace products that turn ambiguous system problems into clear user value and measurable business impact.

## 4. Copilot repositioning

The copilot should stay, but it should no longer be the first thing the page is asking the user to do.

### Suggested section title
**Explore the portfolio with the copilot**

### Suggested section copy
Use the copilot to jump directly into AI experience, strongest outcomes, case studies, leadership scope, or marketplace work.

### Suggested input placeholder
Ask about AI search, marketplace growth, fintech systems, leadership scope, or strongest case studies.

## 8. What should visually happen first

The ideal reading order for a first-time visitor should be:

1. positioning
2. proof
3. featured cases
4. then the copilot

That sequence will convert better than leading with an interactive assistant.
